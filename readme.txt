Import, export, and sync remote templates for Bricks Builder from CSV files, JSON files, or Google Sheets.

== Description ==
 Admin Menu Width is a powerful plugin designed to streamline the process of managing remote templates for Bricks Builder. With this plugin, you can easily import templates from CSV or JSON files, sync templates from Google Sheets, and export your existing templates for backup or transfer to another site.

Key features include:

* Import templates from CSV files
* Import templates from JSON files
* Sync templates directly from Google Sheets
* Export templates to CSV format
* Export templates to JSON format
* Reset all remote templates with a single click
* User-friendly admin interface

This plugin is perfect for Bricks Builder users who need to manage large numbers of templates or frequently update their template library.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/admin-menu-width` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Bricks Templates->Import/Export screen to configure the plugin and use its features

== Frequently Asked Questions ==

= Can I use this plugin with page builders other than Bricks? =

No, this plugin is specifically designed to work with Bricks Builder and is not compatible with other page builders.

= Is there a limit to how many templates I can import? =

There is no hard limit set by the plugin, but very large imports may be limited by your server's configuration (max upload size, execution time, etc.).

= Can I schedule automatic syncs with Google Sheets? =

Currently, the plugin does not support scheduled automatic syncs. You need to manually trigger the sync from the admin interface.
